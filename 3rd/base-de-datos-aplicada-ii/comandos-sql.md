- SELECT INITCAP(NOMBRE_EMP) FROM EMPLEADO; : Le da mayuscula a la primera letra.

```sql
SELECT 
    NUMRUT_CLI AS RUT, 
    DVRUT_CLI AS DV,
    TO_CHAR(NUMRUT_CLI, '999g999g999') ||'-'
    || DVRUT_CLI AS RUT_CLIENTE,
    RENTA_CLI,
    TO_CHAR(RENTA_CLI, '$999g999g999') AS SUELDO,
    INITCAP(NOMBRE_CLI ||' '|| APPATERNO_CLI) AS NOMBRE
FROM CLIENTE
ORDER BY RENTA_CLI DESC
;
```
### CASO 2:

```sql
SELECT 
    NUMRUT_CLI AS NUMERO_RUT, 
    DVRUT_CLI AS DIGITO_VERIFICADOR,
    INITCAP(APPATERNO_CLI ||' '|| APMATERNO_CLI ||' '||NOMBRE_CLI) AS NOMBRE_CLIENTE,
    TO_CHAR(RENTA_CLI, '$999g999g999') AS RENTA,
    FONOFIJO_CLI AS TELEFONO_FIJO,
    CELULAR_CLI AS CELULAR
FROM CLIENTE
ORDER BY APPATERNO_CLI 
;
```
