La web funciona con un modelo cliente-servidor. El cliente navegador web envia slicitudes al servidor y el servidor responde con el contenido solicitado

HTML: Lenguaje de marcado ara crear y estructurar paginas web. Utiliza principalmente los metodos HTTP GET y POST.

HTTP: HyperText Transfer Protocol: protocolo utilizado por la web para la transferencia de documentos y datos.

codigos internet gatitos

Metodos HTTP: Acciones que se puede realizar sobre recursos en una arquitectura web.

GET: solicitar datos del servidor
POST: enviar datos al servidor
PUT: reemplazar por completo un recurso existente
PATCH: actualizacion parcial de un recurso existente.
DELETE:  elimina un recurso del servidor
HEAD: a diferencia del GET devuelve solo los encabezados de la respuesta
OPTIONS: solicitar informacion sobre las opciones de comunicacion disponibles.

Respuestas HTTP: Mensajes enviados por el servidor en repsuesta a una solicitud HTTP
