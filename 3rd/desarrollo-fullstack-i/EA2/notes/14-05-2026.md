### Logs

@SLF4J Simple Logging Facade for Java + Logback: Permiten registrar eventos y errores del sistema,muy usados en microservicios y observabilidad
 - Una API de logging, define como escribir logs
 - No guarda logs, solo define la interfaz
Logback: Motor de logging por defecto en Spring
 - Recibe logs desde slg4j
 - Permite configurar archivos json, niveles y formatos
 - Se configura en logback-spring.xml

 codigo Java -> slf4j -> logback -> consola/archivo/json/elk

Dependencias: logstash-logbacj.encoder
